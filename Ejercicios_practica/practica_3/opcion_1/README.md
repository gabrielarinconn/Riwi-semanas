Mini proyecto "Gestor de Inventario Multiformato"
- Objetivo General
- Crear una aplicacion modular en python que gestione un inventario simple
- Registrar productos, editarlos, eliminarlos, 
- Guardar y leer informacion desde: TXT, JSON, CSV
- Organizar el proyecto usando funciones(una por accion)
- Usar buenas practicas: nombres claros, modularidad, manejo de errores, try/except, validaciones
